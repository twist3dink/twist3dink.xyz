# Contributing to twist3dink.xyz

I just want to say **thank you** for wanting to add to my first project!  
Any and all of your **contributions, big or small**, are very much valued!  
I look forward to learning, growing with you all, and being part of something bigger than myself.

> **Security that is enforced doesn't feel heroic, but if you stay ready, so you never have to get ready.**

## Validation Requirements Before Submitting
1. All PRs must pass the CI checks to be merged.
2. Use of `innerHTML` will trigger a **force review** (grep it: `grep -n innerHTML src/` and comment usage in your PR).
3. Run validators:
   - [W3C HTML Validator](https://validator.w3.org/) (upload file/URLâ€”fixes broken markup).
   - [W3C CSS Validator](https://jigsaw.w3.org/css-validator/).
   - [JS Hint](https://jshint.com/) for code quality.
4. Be respectful and follow the [CoC](https://github.com/twist3dink/twist3dink.xyz/blob/main/CODE_OF_CONDUCT.md).
5. Keep contributions **concise, focused, and well-documented**.

**Questions?** Open an issue using a template from [`/academy/flipper-zero/templates`](https://github.com/twist3dink/twist3dink.xyz/tree/main/academy/flipper-zero/templates) (or root `/templates`).
