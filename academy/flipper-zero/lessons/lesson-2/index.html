Lesson 2: Navigating the Flipper Interface

Estimated time: 20 minutes | XP Reward: 50

The Main Menu

When you power on your Flipper Zero, you're greeted by the home screen featuring your animated dolphin companion. Press any directional button or OK to access the main menu, which serves as your gateway to all of Flipper's capabilities.

The main menu is organized into distinct sections, each representing a different wireless technology or functionality:

125kHz RFID

Low-frequency RFID operations for reading and emulating common access cards, key fobs, and pet identification tags. This frequency is widely used in building access control and animal tracking.

NFC (Near Field Communication)

High-frequency 13.56MHz operations for contactless cards, including payment cards, transit passes, and modern access control systems. NFC supports more complex protocols and data structures than 125kHz RFID.

Sub-GHz

Radio frequency communication below 1 GHz, used by garage door openers, car key fobs, wireless doorbells, weather stations, and many other remote control devices. This is one of Flipper's most versatile modules.

Infrared

IR remote control emulation for TVs, air conditioners, projectors, and other consumer electronics. Flipper can learn new remotes and comes with an extensive database of universal remote codes.

iButton

1-Wire protocol devices, commonly used for physical access control systems, particularly in commercial and industrial settings. These look like small metal buttons that touch a reader.

Bad USB

USB Human Interface Device (HID) emulation, allowing Flipper to act as a keyboard or mouse. This enables automated keystroke injection for penetration testing and system automation.

U2F (Universal 2nd Factor)

Two-factor authentication key emulation for securing online accounts. Flipper can store and use U2F credentials for enhanced account security.

GPIO

General Purpose Input/Output access for connecting external hardware, sensors, and custom projects. This opens up endless possibilities for hardware hacking and IoT experimentation.

Navigation Tip: The main menu wraps around - if you keep pressing down, you'll eventually circle back to the top. The interface is designed for one-handed operation with muscle memory in mind.

Button Functions

Mastering the button layout is essential for efficient navigation. Flipper's control scheme is intuitive but has some nuances worth understanding:

The Directional Pad (D-Pad)

  • Up/Down: Navigate through menu items, scroll through lists, adjust values
  • Left/Right: Switch between tabs, change settings, adjust parameters
  • OK (Center Press): Select menu items, confirm actions, interact with objects

The Back Button

Located below the screen, this button has multiple functions depending on context:

  • Single press: Return to previous menu or cancel current action
  • Long press (home screen): Quick access to settings and system information
  • Long press (3 seconds when off): Power on the device
  • Long press (3 seconds when on): Power menu with shutdown and reboot options

Long Press Behaviors

Many menu items support long-press actions that provide additional options or shortcuts. For example, long-pressing on a saved file often brings up options to rename, delete, or get information about it.

Pro Tip: When in doubt, press Back to return to a familiar screen. The interface is designed to be forgiving - you won't accidentally damage anything or lose data by exploring.

The Dolphin

Your virtual companion isn't just decoration - it's an integral part of the Flipper Zero experience and a clever gamification system that encourages exploration and learning.

Leveling Up

The dolphin gains experience points (XP) as you use your Flipper. Every action you take - reading an RFID card, capturing a Sub-GHz signal, using the IR remote - earns XP. As you accumulate XP, your dolphin levels up and unlocks new animations and expressions.

Animation Changes

Different dolphin levels feature unique idle animations, reactions, and behaviors. Higher-level dolphins have more personality and interactive elements. This serves as a visual representation of your growing expertise with the device.

Interactive Features

Press OK on the home screen to pet your dolphin - it will react with happy animations. The dolphin also responds to certain actions throughout the interface, providing subtle feedback on what you're doing.

Easter Eggs

The Flipper community has discovered various hidden features and special dolphin behaviors. Exploring the interface with curiosity often reveals these delightful surprises. We won't spoil them all - discovering them yourself is part of the fun!

Fun Fact: The dolphin's design was inspired by the idea that dolphins are playful, intelligent creatures that communicate through sound waves - much like how Flipper Zero communicates through radio waves!

Settings Menu

Access settings by long-pressing the Back button from the home screen, or by navigating to the Settings option in the main menu. Understanding these options helps you customize Flipper to your preferences and keep it running smoothly.

System Settings

  • Device Info: View firmware version, hardware revision, serial number, and storage information
  • Region: Set your geographic region to comply with local radio frequency regulations
  • Date & Time: Configure time settings (important for certain protocols and logging)
  • Auto Lock: Set automatic screen lock timeout for privacy
  • LCD Settings: Adjust brightness and screen timeout to balance visibility with battery life

Bluetooth Configuration

Flipper Zero includes Bluetooth Low Energy (BLE) for wireless connectivity:

  • Pair with the official mobile app for remote control
  • Transfer files wirelessly between Flipper and smartphone
  • Access remote logging and debugging features
  • Enable or disable Bluetooth to save battery when not needed

Display Options

Customize the visual experience:

  • Theme: Choose different color schemes or animations
  • Clock Display: Toggle clock visibility on home screen
  • Notifications: Configure how alerts and messages appear

Storage Management

Monitor and manage your SD card storage:

  • View total and available storage space
  • Format SD card (caution: erases all data)
  • Browse internal vs external storage
  • Manage save files and databases

Firmware Updates

Keep your Flipper current with the latest features and security patches:

  • Update Channel: Choose between stable releases or development builds
  • Update via SD Card: Manual firmware installation from update files
  • Update via App: Wireless updates through the mobile application
Important: Never interrupt a firmware update in progress. Ensure your battery is above 30% before starting an update to prevent potential issues.

File Browser

Flipper's file browser is your window into the device's storage system, where all captured signals, saved configurations, and custom files are stored.

SD Card Navigation

The file system is organized into clear folders:

  • /subghz/ - Captured and saved Sub-GHz signals
  • /nfc/ - NFC card dumps and emulation files
  • /rfid/ - 125kHz RFID card data
  • /infrared/ - IR remote control files
  • /ibutton/ - iButton key data
  • /badusb/ - Bad USB script files
  • /apps/ - Third-party applications and plugins

File Management

Navigate through folders using the D-pad. When you select a file, you'll see options to:

  • Open/Run: Execute or emulate the file
  • Info: View file details and metadata
  • Edit: Modify compatible file types
  • Rename: Change the file name
  • Delete: Remove the file (with confirmation)

Supported File Types

Flipper works with various file formats specific to each module:

  • .sub - Sub-GHz signal files
  • .nfc - NFC dump files
  • .rfid - RFID card data
  • .ir - Infrared remote files
  • .txt - BadUSB scripts (Ducky Script format)
  • .fap - Flipper Application Package (third-party apps)
Organization Tip: Create subfolders within each module's directory to organize your files by category. For example, /subghz/cars/, /subghz/garage-doors/, etc. This makes finding specific captures much easier as your library grows.

Quick Tips

Here are some essential best practices to get the most out of your Flipper Zero:

Battery Management

  • Flipper uses a 2000mAh rechargeable lithium battery that lasts 3-7 days with typical use
  • The device can operate while charging via USB-C
  • Disable Bluetooth when not in use to extend battery life significantly
  • Lower screen brightness reduces power consumption
  • The battery indicator shows remaining charge - plan ahead for extended field work

SD Card Recommendations

While Flipper has internal storage, an SD card dramatically expands its capabilities:

  • Size: 8-32GB is plenty for most users; supports up to 128GB
  • Speed: Use Class 10 or UHS-I cards for best performance
  • Format: FAT32 file system is required
  • Quality: Use reputable brands to avoid corruption and read errors

Firmware Update Process

Keeping firmware current ensures you have the latest features and security improvements:

  1. Check current version: Settings → Device Info
  2. Visit official website: flipperzero.one or use the mobile app
  3. Download firmware: Choose stable or development channel
  4. Install update: Via mobile app (easiest) or SD card
  5. Verify installation: Check version after reboot
Custom Firmware: The community has created alternative firmware options like Unleashed and Xtreme that offer additional features. These are safe to use but void some warranties. Research thoroughly before switching from official firmware.

Practice Exercise

Now that you understand the interface, take 10 minutes to explore your Flipper:

  1. Navigate through each main menu option and observe what's available
  2. Long-press the Back button to access quick settings
  3. Check your firmware version in Settings → Device Info
  4. Browse the file system and familiarize yourself with the folder structure
  5. Pet your dolphin on the home screen and watch its reaction
  6. Adjust your screen brightness to a comfortable level
  7. Enable or configure Bluetooth if you have the mobile app

Don't worry about understanding what each module does yet - we'll cover the functionality in detail in upcoming lessons. For now, focus on becoming comfortable with navigation and menu structure.
